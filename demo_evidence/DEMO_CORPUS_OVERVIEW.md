# GeoTrace Demo Corpus

This folder is organized into analyst-friendly scenarios.

## Cases

- `case_01_gps_original/` — GPS/original-media style import path.
- `case_02_screenshot_map/` — map screenshot, OCR labels, route/map intelligence.
- `case_03_hidden_payload/` — hidden-content and pixel/stego triage.
- `case_04_timeline_conflict/` — timestamp contradiction and timeline validation.
- `case_05_ctf_landmark/` — CTF-style landmark/location reasoning.

## Quarantine

Legacy duplicates and malformed demo artifacts were moved to `_quarantine_legacy_duplicates/`.
Keep them only for parser robustness tests; do not present them as polished demo evidence.
