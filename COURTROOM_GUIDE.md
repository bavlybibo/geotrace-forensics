# GeoTrace Forensics X — Courtroom Guide

GeoTrace is a triage and reporting assistant, not a replacement for examiner judgment.

## Stronger evidence

- Native EXIF GPS from original media.
- Verified source/working-copy hashes.
- Consistent timeline from original device or platform source.
- Multiple independent anchors agreeing on location/time.

## Weaker evidence

- Filename-derived timestamps.
- Screenshot-derived coordinates.
- OCR-only place labels.
- Heuristic map/landmark matches.

## Recommended language

Use “supports,” “suggests,” or “is consistent with” for derived leads. Reserve “confirms” for independently verified source-side evidence.
